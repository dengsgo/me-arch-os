## Feature

- 基于 Arch Linux ，提供开箱即用的桌面发行版  
- 无冗余软件，只附加最低的基础软件，满足定制化需求    
- 完全兼容 Arch Linux，不篡改发行版 ID  
- 专注于 Kde Plasma 桌面  
- 特定国家和地区特殊优化，以更好适应用户习惯
